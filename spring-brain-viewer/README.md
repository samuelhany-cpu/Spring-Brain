# spring-brain-viewer

> **Planned for a future milestone. Not implemented in Milestone 0.**

This module will contain the interactive React-based architecture viewer for Spring Brain.

## Planned Tech Stack

- React
- TypeScript
- Vite
- React Flow
- Tailwind CSS

## Planned Features

- Interactive graph visualization of Spring Boot architecture
- Endpoint lifecycle view (Route → Controller → Service → Repository → Entity)
- Diagnostic panel with severity filtering
- Node detail panel with source file links
- Search and filter by node type
